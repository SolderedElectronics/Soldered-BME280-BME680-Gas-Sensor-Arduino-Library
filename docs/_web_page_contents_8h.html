const char MAIN_page [] PROGMEM

This character string "MAIN_page" is loaded into program memory and contains the web page that is served when a browser calls up the site. It uses the javascript from https://www.chartjs.org to generate a page with a dynamic chart and lists the data in tabular form underneath as well.

Since this is used for demonstration purposes it has been kept basic and simple.
